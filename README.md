# PathFinder - Career Guidance Web App

**PathFinder** is a web-based application that helps students identify suitable career paths and courses based on their interests, academic preferences, and personality traits. By answering a simple quiz, users receive a tailored course and career suggestion.

---

---

## 🧩 Features

- Responsive and user-friendly interface.
- 10-question quiz to capture user preferences.
- Smart data-matching from a comprehensive dataset.
- Suggests a course and career based on input.
- Fallback suggestion when no perfect match is found.
- Input validation ensures users complete all questions.
- Professionally styled with a clean purple-gradient theme.
